# Generate Lineups for NFL, NBA or NHL Contests on DraftKings
***
1. Enter lineups to a contest, and download the CSV file from DraftKings.

2. Replace the DKEntries.csv in this sample with the downloaded file.

3. Edit the points.csv with players from the contest you entered, and give them projected points.

4. Run the application. Lineups will be generated automatically, and be displayed in the console. 

5. Click the save button to create a CSV file that is ready to be uploaded.
